# LangChain Chatbots Project
This repository contains multiple Python-based chatbot implementations using LangChain and Hugging Face models.
The project demonstrates prompt handling, memory-based conversations, and agent-driven chat applications.

# Features
-Basic chatbot using LangChain
-Chatbot with conversation memory
-Agent-based chat application
-Hugging Face prompt & message demos
-Streamlit-based chat UI

# Technologies Used
-Python
-LangChain
-Hugging Face Transformers
-Streamlit
-Git & GitHub

# Project Files
-langchain_chatbot.py – Basic chatbot implementation
-agent_chat_app.py – Agent-based chatbot
-session_memory_chat.py – Chatbot with memory
-streamlit_hf_chat.py – Streamlit web interface
-requirements.txt – Project dependencies

# How to Run
-pip install -r requirements.txt
python langchain_chatbot.py

For Streamlit app:
streamlit run streamlit_hf_chat.py

Create a .env file and add:
HUGGINGFACE_API_KEY=your_api_key_here
